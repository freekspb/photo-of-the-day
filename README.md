Описание
-----------

Суть приложения сведена к простейшему минимуму. C наступлением нового дня приложение раз в час проверяет наличие новой фотографии. Если доступна новая фотография, она закачивается и устанавливается в качестве обоев. 
Также можно указать тег(ключевое слово) по которому будут выбираться обои и при желании установить обновление обоев каждый час. Экспериментируйте с тегами: природа, кошки, собаки, море, небо, цветы, девочки, мальчики, знаменитости, певцы, актеры, автомобили, кино, молния, игры и многими другими и Вы получите замечательные обои на Ваш вкус.

Поддерживаемые сервисы:
- Яндекс.Фотки
- Flickr
- National Geographic
- Earth Shots (обновление днем)

Подстраивать размеры из исходной фотографии совершенно не нужно. Приложение вычисляет все самостоятельно, так что при выборе фотографии необходимо только согласиться с предложенным вариантом и далее наслаждаться результатом.
Также вы сможете сохранить понравившиеся обои на свой телефон с помощью виджета.

Возможности виджета:
- Открытие галереи сохраненных обоев
- Сохранение понравившихся обоев в галерею
- Следующий источник обоев
- Открытие полных настроек Фото дня
